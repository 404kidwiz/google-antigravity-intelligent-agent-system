## üîß Troubleshooting

### Expert Not Appearing
**Issue**: Wrong expert responding
**Solution**: 
- Use specific technology names
- Restart Antigravity
- Test with: "React component" not just "component"

### Not Switching Experts
**Issue**: Stuck with same expert
**Solution**:
- Use different technology keywords
- Start new conversation thread
- Explicitly mention new domain: "Now I need Python help..."

### Agency Coordination Not Working
**Issue**: Only one expert responds
**Solution**:
- Use project-level requests: "Build a full-stack app"
- Be specific about all domains involved
- Try: "I need frontend, backend, and database"
